Storebrand.ny - Version 0.87

Web manual as of 19.10.2015

What's new? The corporate part is now a part of the menu, footer and login, mainly changes made in javascript.

  • Video fix. The buffring setting is changed to be able to restart the video
  • Login for corporate pages is added

Here's the code you'll want to include to start playing with this version:

 <link rel="stylesheet" type="text/css" href="//elements.storebrand.no/storebrand.ny/0.87/css/stb-all.min.css">
 <script type="text/javascript" src="//elements.storebrand.no/storebrand.ny/0.87/javascript/stb-all.min.js"></script>

gzipped versions: smaller, faster, push to master!

 <link rel="stylesheet" type="text/css" href="//elements.storebrand.no/storebrand.ny/0.87/css/stb-all.gz.css">
 <script type="text/javascript" src="//elements.storebrand.no/storebrand.ny/0.87/javascript/stb-all.gz.js"></script>
